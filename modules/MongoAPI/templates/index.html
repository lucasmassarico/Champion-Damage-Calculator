Endpoints
  • /champions

    GET: Request all champions statistics

  • /champions/{champion_id}

    GET: Request a specific champion statistics

    POST: Create a new champion

      [header] Login Token

      [required] name: string

      [required] resource: string

      [required] attackType: string

      [required] adaptiveType: string

      [required] stats: dict

      [required] abilities: dict

    PUT: Create or update a champion

      [header] Login Token

      [required] name: string

      [required] resource: string

      [required] attackType: string

      [required] adaptiveType: string

      [required] stats: dict

      [required] abilities: dict

    DELETE: Delete a specific champion

      [header] Login Token

  • /users/{username}

    Request a specific user

  • /register

    POST: Create a new user to access data in BD

      [header] Login Token

      [required] username: string

      [required] password: string

      [default: read]level: string

  • /login

    POST: Access user to generate a Token

      [required] username: string

      [required] password: string

  • /logout

    POST: Logout user

      [header] Login Token
